Detailed Description

Manages creation and manipulation of a Spotify user profile.

Parameters
debugAn optional Boolean parameter specifying whether or not to run in verbose mode
logfileAn optional filename for a test file containing a user profile vector

The User class manages the local representation of a Spotify user, as aggregated from a dataset composed of the songs in their Spotify library. A profile vector represents the average values for each of the features we track and recommend with, and is represented locally in a text file (soon to be database)

processProfile

Reads in a user profile from a local logfile

Parameters
filenameThe text document which contains the user profile vector information.
Returns
void

This method opens the file passed in and parses each line into the User.profile member. it is assumed that the filename passed in refers to a valid and accesible file in the CWD and that the file is properly formatted. We used {{a,,b,,c}} to denote lists, and each line is parsed as an element of the profile vector.

def REKTUser.User.saveStatus (   self)

saveStatus

Saves the current user profile vector to disk

Returns
void

This method aggregates the new values in the mutator functions and writes them to the user profile logfile, preserving the state for future access by our program. Each element in the profile vector is written to its own line. Values are written directly, and lists are written as {{a,,b,,c}} to represent the list [a,b,c].
