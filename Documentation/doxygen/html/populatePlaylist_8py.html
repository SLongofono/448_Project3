Detailed Description

Gathers, vets and creates a playlist for songs.

This script strings together the process of gathering songs to process, computing the weighted differnce from average, and adding the least different songs to a playlist on the user's Spotify account. This script is called after updating the user's database in the run.py script

filterDifference

Simple helper function to filter out dictionary entries with differences beyond a threshold value

Parameters
songsDictA dictionary of the form Spotify track ID: weighted feature differences
thresholdAn optional parameter specifying the upper bound for the cum of the weighted feature differences
Returns
A list of track IDs associated with songs that had total difference less than the threshold

This is a helper method used to filter a dictionary type. The input dictionary contains the weighted feature-wise difference of each track ID, the sum of which must be below the threshold for the song to be considered a match.
