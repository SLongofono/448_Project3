Detailed Description

Script responsible for gathering, vetting, and adding songs to the user playlist.

This script is designed to be run periodically to generate new recommendations for the user. The script will instantiate a user, determine if the profile needs to be updated, gather and filter songs, and finally load the songs that meet our criterion into a special Spotify playlist.
